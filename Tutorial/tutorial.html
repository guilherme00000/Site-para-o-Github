O que é PHP?

PHP é uma linguagem de programação voltada para o desenvolvimento de aplicações para a web e para criar sites, favorecendo a conexão entre os servidores e a interface do usuário.
Entre os fatores que fizeram o PHP se popularizar bastante é possível apontar, principalmente, o fato de ser em código aberto. Isso significa que qualquer um pode realizar mudanças na estrutura dessa linguagem.
Na prática, isso representa duas coisas importantes: primeiramente, por ser de código aberto, não há restrições de uso ligadas a direitos.
O usuário pode usar PHP para programar em qualquer projeto e, depois, comercializá-lo tranquilamente. Além disso, a outra vantagem é a possibilidade de melhorias constantes no código, algo que a comunidade de programadores costuma fazer.

Como baixar o PHP em sua máquina

Para fazer o download do PHP em seu computador, acesse o site: https://www.php.net/downloads.php

Na tela inicial, clique em: Windows download

Depois de ter clicado em Windows download, acesse a primeira opção zip, que será a extensão usada para fazer o download do php.

Depois de ter selecionado essa opção, o download do arquivo começará, apenas aguarde que finalize para continuar o passo-a-passo.

Depois do download finalizado, acesse o Explorador de Arquivos do seu computador e desça até a opção dowsnloads, ache o nome do arquivos e clique com o botão direito em cima dele e, selecione a opção Extrair para...

Quando terminar a extração do arquivo, pegar a pasta e por lá dentro do Disco C:

Criando um arquivo executavél

Depois que a pasta estiver dentro do Disco C:, é necessário criar um arquivo de texto executavél. Para isso, clique com o botão dereito em um espaço vazio dentro da própria pasta, e na janela que se abrir de opções, siga as indicações das setas vermelhas.

Feito isso, vamos editar o documento de texto e salva-ló da maneira correta, para se criar um arquivo executavél apartir dele.
Lembrando que o arquivo tem que ser salvo com a extensão .php
Siga o exemplo do script que está edentificado na imagem, para criar um script simples e vamos salva-ló como: exemplo.php

Para salvar o documento, clique em "Arquivo" e depois em "Salvar Como..."

Na tela que se abrir, na opção Nome, nomear o arquivo como "exemplo.php" e em Tipo selecionar a opção "Todos os arquivos" e por fim salvar.

Se tudo der certo, você terá uma arquivo executavél como esse que está aparecendo na tela indicado pela seta vermelha.

Como executar o arquivo?

Abra o CMD, usando as teclas de atalho win + R. E no menu suspenso que se abre, digite CMD.

Para executar o arquivo, na tela inicial do CMD, use o comando "c.." que fará você navegar para o diretório anterior, com isso, você acessará o "Disco Local C:", dentro dele faça o comando "cd" + o nome da pasta onde está seu arquivo.

Para executar o arquivo propriamente dito, use está sintaxe: php -S localhost:80 -t "nome da pasta"

Feito isso, agora acesse o navegador de internet, pode ser o Edge, e na barra da pesquisa coloque este comando: localhost/nomedoexecutavél. Depois de enter e o arquivo será executado na página web.

Como baixar e instalar o XAMPP e executar o arquivo

Para fazer o download do XAMPP, acesse este link: https://www.apachefriends.org/index.html

Depois de ter instalado o XAMPP, agora inicie o painel de controle dele, pode ser salvo na barra de tarefas mas, ele pode ser acessado através do CMD também, usando o seguinte comando: C:\xampp\xampp-control.exe. Depois de vizualizar o painel clique em start para inicar o servidor.

Depois de inicar o servidor, agora vamos acessar a pasta "htdocs" que está dentro dos documentos do XAMPP, e lá dentro vamos salvar um arquivo executavél. Neste caso, vamos usar o mesmo documento que foi usando dentro do cmd, para isso basta usar o Ctrl C + Ctrl V para colar o executavél dentro da pasta desejada.


Agora abra o navegor do computador e neste momento digite: localhost/"nome do arquivo". Isso fará com que ele seja executado pelo XAMPP.
